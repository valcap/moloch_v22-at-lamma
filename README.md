# Package of Numerical Weather Prediction model MOLOCH

Version 22 (July) by P Malguzzi and implemented at LaMMA

Compile geo.F90 with informations from geo.inc and static files (to be downloaded separately)
gfortran -o geo.exe geo.F90

Then run:
compila-premoloch
compila-moloch
compila-ppostmol
compila-shf2grib2
